# token-resilience-weekly

代币韧性周报 — 量化哪些代币在大盘承压时展现超额抗跌能力

## 功能

调用 `/token-resilience-weekly TAO,KAITO,HYPE,IO,SPEC,STORY,DRIFT` 后，Claude 会：

1. 拉取目标代币和基准（ETH、SOL）过去一周的 CoinGecko 每日收盘价
2. 计算每日涨跌幅，构建"日期 × 代币"矩阵表
3. 计算周均日涨跌幅，对比基准均值
4. 按韧性得分排序并分级（T1 / T2 / 弱势 / 中性）
5. 输出完整韧性周报，含矩阵表、排名表、分析摘要

**方法论来源**: Eric @Guolier8 的代币韧性周报

**韧性定义**: 大盘反弹时涨得更多，回调时跌得更少

**韧性分级**:
- **T1** 🟢：周均涨跌幅 > 0%（大盘跌的周期还能正收益）
- **T2** 🟡：周均涨跌幅 < 0%，但跌幅明显小于基准
- **弱势** 🔴：跌幅远超大盘（基准均值的 2 倍以上）
- **中性** ⚪：其余

## 前置要求

**必须安装 Antseer MCP 服务器**，它提供以下数据工具：
- `ant_spot_market_structure` — CoinGecko 代币历史价格、搜索、详情

## 安装步骤

### 第 1 步：安装 Antseer MCP

按照 Antseer 官方文档安装并配置 MCP 服务器，确保在 Claude Code 中能看到 `ant_spot_market_structure` 工具。

### 第 2 步：克隆 skill

将此目录复制到你的 Claude Code skills 目录：

```bash
cp -r token-resilience-weekly ~/.claude/skills/
```

### 第 3 步：运行配置脚本

```bash
cd ~/.claude/skills/token-resilience-weekly
chmod +x setup.sh
./setup.sh
```

脚本会自动检测你的 Antseer MCP UUID 并写入 `SKILL.md`。
如果自动检测失败，按提示手动粘贴 UUID 即可。

> **如何手动找 UUID**：在 Claude Code 中输入 `/tools`，找到工具名
> `mcp__<UUID>__ant_spot_market_structure`，复制 `<UUID>` 部分。

### 第 4 步：使用

```
/token-resilience-weekly TAO,KAITO,HYPE,IO,SPEC,STORY,DRIFT
/token-resilience-weekly AAVE,UNI,LINK,ARB,OP
/token-resilience-weekly BTC,ETH,SOL,AVAX,MATIC
```

也可以用自然语言触发：
```
TAO,KAITO,HYPE 的韧性周报
哪些币抗跌？帮我看看 TAO KAITO HYPE
本周代币韧性排名
```

## 适用范围

- **适合**: 任何在 CoinGecko 有价格数据的代币
- **默认基准**: ETH + SOL（可通过对话修改）
- **默认周期**: 7 天（可调整）
- **数据粒度**: 日级收盘价（非实时）
